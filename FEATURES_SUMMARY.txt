CUSTOMER FEATURES
================================================================================

✅ SCAN QR BUTTON
   - Location: Main page header (blue button)
   - Mobile: Available in menu
   - Function: Navigate to QR scanner page
   - Icon: QR code icon with "Scan QR" text

✅ QR CODE DISPLAY
   - Shows all 10 table QR codes
   - Each QR displays:
     * Table number (in colored badge)
     * QR code image
     * Status indicator
   - Color-coded: Green (available) / Red (occupied)

✅ TABLE SELECTION
   - Click/tap QR code to select table
   - Real-time availability check
   - Prevents selecting occupied tables
   - Shows success/error alerts
   - Automatic redirect to menu

✅ AUTOMATIC TABLE TRACKING
   - Table number stored in cart
   - Persists across sessions
   - Linked to all orders
   - Cleared when order completes

✅ VISUAL STATUS INDICATORS
   - Available: Green checkmark (✓)
   - Occupied: Red warning (⚠️)
   - Processing: Blue spinner (⟳)
   - Disabled state for occupied tables

  4. Start ordering

  2. Log in
  3. See Table Status Panel
  4. Monitor occupancy in real-time
